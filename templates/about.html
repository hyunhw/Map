About

This application uses the Indego Bike Share API to retrieve live information on bike share stations in Philadelphia, PA. If you click on the marker, it will show how many bikes are availble, how many docks are available, and (soon coming) whether or not the station will become full/empty over the next hour.
This site is powered by flask with a custom designed html/css template and uses the Google Maps Javascript API, and the backend uses the pandas data analysis library for python.
