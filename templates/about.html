About

This application uses the Indego Bike Share API to retrieve live information on bike share stations in Philadelphia, PA.
If you click on the marker, it will show how many bikes and docks are availble at that station.

This application also uses the weather API provided by Weather Underground to retrieve the current weather information to predict how busy the stations will be today.

This site is powered by flask with a custom designed html/css template and uses the Google Maps Javascript API.
The backend uses the pandas data analysis library and the scikit-learn machine learning library for python.
